# Marketing Attribution Analytics

## Project Overview

This project focuses on analyzing marketing campaign performance across different channels to measure effectiveness, profitability, and return on investment. The analysis helps identify high-performing campaigns and optimize marketing spending through data-driven insights.

## Problem Statement

Organizations invest heavily in marketing campaigns but often struggle to determine which channels generate the best results. This project evaluates campaign performance using key marketing metrics and visual analytics.

## Dataset Information

The dataset contains marketing campaign details including:

* Campaign ID
* Start Date
* End Date
* Channel
* Impressions
* Clicks
* Leads
* Conversions
* Cost (USD)
* Revenue (USD)
* ROI
